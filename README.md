# Tetris Babylon | Made by Firdeus Kasaj

This is a project of creating a Tetris Game in Babylon Js using Typescript.<br>
Still in progress


<h2>How to install</h2>
npm i <br>
npm run build<br>
npm run start<br><br>

And then go to your local host

http://localhost:8080/
